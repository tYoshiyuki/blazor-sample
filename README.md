# blazor-sample
BlazorでCRUDをするサンプル

# Feature
- .NET Core
- Blazor

## Note
- Blazorを使ったCRUDのサンプルです。
- ASP.NET MVCっぽく ViewModel を利用しデータのやり取りを行います。
- PWAに対応しています。
- データストアにLocal DBが必要です。
